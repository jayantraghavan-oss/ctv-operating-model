# CTV GTM Alignment Doc — Key Insights for Reporting Layer

## Core Framework: 4-Stage Pipeline
1. **Pre-Pitch / Lead Identification** — Global CTV Tracker as SSOT, leads by app promo plans + budget + CTV readiness
2. **Pitch** — Standardized decks (CTV-to-App, CTV-to-Web), quarterly sales training, pitch learnings captured
3. **Campaign Onboarding & Health** — Intake form, health monitoring, customer sentiment in xFN Exec Weekly
4. **Test → Scale** — Convert 4-week tests to evergreen/scaled spending

## Key Data Points to Surface in Reporting
- **Pipeline by stage**: Pre-Pitch → Pitch → Onboarding → Test → Scale
- **Customer personas**: CTV-experienced (performance), CTV-experienced (branding), CTV-new
- **Regional breakdown**: AMER, EMEA, APAC (US supply focus)
- **Pitch learnings**: Objections, customer needs, expectations (from Gong)
- **Campaign health**: Customer sentiment, campaign issues, next steps
- **Test-to-scale conversion**: 4-week test → evergreen criteria
- **Churn analysis**: Right fit vs wrong fit, gap identification, revisit plans
- **Success codification**: Key drivers, case study pipeline, replicable lessons
